# FarmaFácil

Site estático de exemplo para a farmácia online "FarmaFácil".

## Conteúdo

- `index.html` - página inicial
- `cadastro.html` - cadastro de usuários
- `login.html` - página de login
- `estoques.html` - gestão de estoques
- `noticias.html` - notícias
- `telemedicina.html` - telemedicina
- `contato.html` - contato
- `acompanharpedido.html` - acompanhamento de pedidos
- `resultados.html` - resultados
- `style.css` - estilos do site
- `main.js` - script principal
- imagens no diretório raiz

## Como usar

1. Abra `index.html` no navegador para visualizar o site localmente.
2. Para usar no GitHub:
   - Crie um repositório no GitHub.
   - Adicione o remote: `git remote add origin <URL do repositório>`.
   - Faça push: `git push -u origin main`.

## Observações

- Este projeto é um site estático sem backend.
- O GitHub Pages pode ser usado para hospedar diretamente o site.
